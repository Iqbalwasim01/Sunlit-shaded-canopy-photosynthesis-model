Abbrevations or meaning:

File.length= in this input, you either paste the number of rows of your data file (e.g. five for five days or observations) or can use "nrow(name of file)" which automatically counts the number of rows in your data file

No.kinetic.sets= If testing more than one Rubisco species (or kinetic sets) place the number here e.g. 10 for 10 kinetic data sets for 10 species or leave as 1 for one set.

LAI= Leaf area index (m-1 m-2); Proxy for the size of the plant

crop.type="Wheat", "Rice", "Maize" or "Sugar beet" 

leaf.angle="Wheat", "Rice", "Maize" or "Sugar beet"  "

pathway= "C3" or "C4" 

Tair= Air temperature (degrees celsious)

Humidity= Humidity (%)/100 or ratio

Precip= Precipation unit area 

SWC= Soil water content (ratio)

Rn= Net radiation (W m-1) If not available can be set NA

H= Latent flux (W m-2) If not available can be set NA

LE= Latent flux (W m-2) If not available can be set NA

LMA= Leaf mass area (g m-2)

SZA= Solar zenith angle 

SW_down= Solar radiation (W m-1)

Ca= Atmospheric CO2 (umol CO2 mol-1)

Sensor.height= (meters)

Kcat, Kc, Sc/o,= (s-1), (mM), (mol mol-1)

KcHA, VmaxHA, GamHA= Heat activation values for each Rubisco kinetic (KJ)
